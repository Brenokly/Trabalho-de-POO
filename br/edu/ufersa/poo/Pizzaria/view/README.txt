Aqui fica o carregamento das telas fxml, uma classe com métodos staticos para carregar as telas.

Cada um dos métodos são responsável por uma tela, a maioria das telas foram dividas em três partes

1 - Primeiro as telas de "listagem", as telas de listagem são responsável	por carregar e exibir os dados do banco de dados
para serem exibir ao usuário

2 - Segundo as telas de "editar" os métodos com o nome editar é porquê são resposáveis por editar os dados que o usuário selecionou na tela de listagem

3 - Terceiro as telas de "cadastro" são responsáveis por inserir novos dados ao banco de dados

as demais telas que não seguem esse padrão são de casos específicos de telas
